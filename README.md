# 🏆 クイズ大会システム (WebRTC 統合版)

ブラウザだけで完結する、クイズ大会用の本格的なスコアボード・システムです。
面倒なサーバー構築は不要。**たった1つのURL**をプロジェクターとスマホで開くだけで、リアルタイムに連動するクイズ環境が完成します。

## 🚀 使い方

### 1. 準備：URLを開く
大会の運営には、以下のURLを使用します。プロジェクター用PCと、操作用スマホの両方でこのリンクを開いてください。

👉 **[ここをクリックしてクイズシステムを開く](./index.html)**
*(※GitHub Pagesで公開している場合は、ご自身の公開URL `https://[ユーザー名].github.io/[リポジトリ名]/` を開いてください)*

### 2. 表示画面（スコアボード）のセットアップ
1. 会場のプロジェクターやモニターに繋いだPCでURLを開きます。
2. 最初の画面で **「📺 プロジェクター用」** をクリックします。
3. 画面上部に **「操作画面でこのIDを入力: (ランダムな英数字)」** が表示されるので、そのIDをメモします。

### 3. 操作画面（コントローラー）のセットアップ
1. 司会者や運営スタッフのスマートフォン等で、同じURLを開きます。
2. 最初の画面で **「📱 スマホ操作用」** をクリックします。
3. 先ほどメモした **ID** を入力し、「接続する」ボタンを押します。
4. 接続が完了すると、両方の画面からID表示が消え、準備完了です！

---

## ✨ 主な機能と操作方法

* **表示人数の一括変更**: スマホ画面上部のドロップダウンから、2人〜8人まで即座に表示枠を切り替えられます。
* **「〇 正解」「✕ 誤答」ボタン**: 自動で「ピンポン！」「ブー！」という音が鳴り、スコアが+1されます。
* **スコアの手動調整 (+/-)**: 変則ルールのために、〇と✕の数を個別に増減できます。
* **状態変更 (通常 / 勝ち抜け / 脱落)**:
  * `勝ち抜け`: 目標ポイントに到達した際などに使用。画面上で「WINNER!」の派手な演出が入ります。
  * `脱落`: ペナルティ上限に達した際などに使用。画面上でプレイヤーがグレーアウトします。
* **プレイヤー名の変更**: スマホ側のテキストボックスを書き換えると、プロジェクター側もリアルタイムで更新されます。

## 🎯 対応可能なクイズルールの例

* **7〇3✕ (ナナサン)**
  * 通常通り 〇 / ✕ ボタンを使用します。
  * 〇が7になったら状態を「勝ち抜け」に、✕が3になったら「脱落」に手動で変更します。
* **アップダウン**
  * 正解でスコア加算、誤答でスコアが0になるルールです。
  * 誤答時は「✕ 誤答」ボタンではなく、〇スコア横の「-」ボタンを使って手動で0点に戻すことで対応できます。
* **スウェーデン方式**
  * 誤答時のペナルティ（✕の数）に応じて、正解時に必要なポイントが変動するルールです。
  * ペナルティ数（✕の数）は記録しておき、スコア（〇の数）を「+」「-」ボタンで自由に調整しながら進行します。

---

## ⚠️ 注意事項

* このシステムはブラウザ同士を直接つなぐ技術 (WebRTC) を利用しているため、**インターネット接続環境が必要**です。
* 音声を鳴らす機能は、ブラウザの仕様により「ユーザーが画面を一度クリックするまで」無効化される場合があります。（本システムでは、最初の「📺 プロジェクター用」ボタンを押したタイミングで自動的に音声が有効化される仕組みになっています）
