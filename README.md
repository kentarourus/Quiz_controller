# 🏆 クイズ大会用 リアルタイムスコアボード

## 🚀 各画面へのアクセス

大会を運営するには、以下のリンクをクリックしてそれぞれの画面を開いてください。

* [📺 **表示画面（スコアボード）を開く**](./display.html)
  ※ プロジェクターやメインモニターなど、観客に見せる画面で開いてください。
* [📱 **操作画面（コントローラー）を開く**](./controller.html)
  ※ 司会者や運営スタッフの手元のスマホ、タブレット、PCなどで開いてください。

---

## ✨ 主な機能

* **完全サーバーレス**: PeerJS (WebRTC) を使用し、ブラウザ同士が直接通信します。
* **スマホからリモート操作**: 司会者や運営スタッフが、手元のスマートフォンからリアルタイムに操作できます。
* **柔軟な人数設定**: 2人〜最大8人まで、表示するプレイヤー数をワンタッチで切り替え可能です。
* **内蔵サウンド**: 「ピンポン！」「ブー！」という音を鳴らせます。
* **あらゆるルールに対応**: スコアの直接編集（＋/－）機能があるため、様々なクイズ形式に対応できます。

---

## 🚀 使い方

### 1. 表示画面（スコアボード）の準備
1. 上記の [📺 表示画面（スコアボード）を開く](./display.html) をクリックして開きます。
2. 画面上部に **「操作画面（スマホなど）でこのIDを入力してください: (ランダムな英数字)」** が表示されます。このIDをメモします。

### 2. 操作画面（コントローラー）の準備
1. 手元のスマートフォンなどで、このページの [📱 操作画面（コントローラー）を開く](./controller.html) をクリックして開きます。
2. 表示画面でメモした **ID** を入力して「接続する」を押します。
3. 接続が完了すると、**両方の画面からID入力要求が消え**、準備完了となります！

---

## ⚠️ 注意事項
* 端末同士の初期接続を行うため、**インターネット接続環境が必要**です。
* 音が鳴らない場合は、表示画面（`display.html`）のどこかを一度クリックしてください。
